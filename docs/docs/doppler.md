## Shunt and Stroke Volume Calculations  

**LVOT Stroke Volume:**  
\[
SV = π (r²) × VTI
\]

**Qp/Qs ratio (for shunts):**  
\[
Qp/Qs = (VTI_{PV} × Area_{PV}) / (VTI_{LVOT} × Area_{LVOT})
\]
  > 1.3 → significant L→R shunt  
  < 0.8 → R→L shunt (rare)
